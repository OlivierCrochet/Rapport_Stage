La Mission

Présentation du stage

Objectif :

Mon objectif durant ces 6 semaines de stage était la mise en place d’une application web permettant d’accompagner les parcours professionnels et les évolutions de carrière. Les utilisateurs principaux de cette application sont les agents du milieu informatique de l’INSEE et les agents RH. Cet Outil doit répondre à trois missions principales :
- Connaître son environnement métier
- Se situer et se décrire dans l’organisation Insee et les référentiels RH
- Se projeter en fonction de sa situation dans l’organisation INSEE et se préparer à évoluer dans son parcours professionnel.

Travail Réalisé :

Analyse des technologies utilisées :

Hibernate Hibernate est un Framework open source de type ORM (Object relational Mapping) et permet de lier/mapper un objet java avec une table dans une base de données grâce à un fichier de mapping. Hibernate peut s’occuper de la création des tables, comme c’est le cas pour l’application que nous avions à développer. Pour assurer la transformation d’une classe java en table de base de données, Hibernate propose deux options différentes. D’une part, on peut utiliser un fichier XML pour déclarer les métadonnées de mapping. D’autre part, on peut utiliser des annotations intégrées dans le code java. Cette dernière options présente de nombreux avantages, notamment pour la clarté et les manipulations de fichiers. C’est donc celle-ci qui a été utilisée. - permet de taper le moins de code possible

Spring MVC (springboot)

Spring MVC est un Framework gratuit et libre qui possède l’avantage intéressant de l’injection de dépendance, mais offre aussi une meilleure structuration de code. Pour cela, on dispose de classes permettant de traiter les requêtes de l’utilisateur ; les Controller. Chaque Controller est représenté par une classe qui contient les méthodes décrivant les actions à effectuer pour traiter les requêtes des utilisateurs. Lors de mon arrivée, l’application comptait 6 Controller, et il a fallu ensuite en rajouter 2. - Permet de taper le moins de code possible

Plotly.Js

Il s’agit d’une librairie JavaScript qui offre la possibilité de créer de nombreux graphiques interactifs de forme diverses. Nous nous en sommes servis pour ajouter un graphique permettant d’accéder, depuis l’emploi-type d’un agent sélectionné, aux emplois-type les plus proches.
